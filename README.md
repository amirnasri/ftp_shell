Installation
================
    pip3 install ftpshell

**Requires:**
- [Python 3.x](https://www.python.org/downloads/)

Usage
================
After the installation, ftpshell script is installed in `/usr/local/bin` and can be 
invoked using:
         
    ftpshell [username[:password]@]server[:port]
